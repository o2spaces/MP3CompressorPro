MP3 Compressor Pro

บีบอัดหลายไฟล์พร้อมกัน · เปลี่ยนชื่อไฟล์แบบกลุ่ม — ใช้งานได้เลยในเบราว์เซอร์

🛠️ ต้องเปิดผ่าน Web Server
เบราว์เซอร์ไม่อนุญาตให้โหลด encoder (lamejs) จาก CDN เมื่อเปิดไฟล์ตรงๆ
ต้องเปิดผ่าน web server ก่อน — ทำได้ง่ายมาก ไม่ต้องติดตั้งอะไร
วิธีที่ 1 — Python (แนะนำ)
Python มาพร้อม Windows/Mac/Linux อยู่แล้ว ไม่ต้องติดตั้ง
python -m http.server 8000
แล้วเปิด Chrome ไปที่ http://localhost:8000
วิธีที่ 2 — VS Code Live Server
1. เปิดโฟลเดอร์ที่มีไฟล์ .html ใน VS Code
2. คลิกปุ่ม Go Live มุมล่างขวา
3. เบราว์เซอร์จะเปิดอัตโนมัติ ✅

🖥️ GPU ช่วยได้แค่ไหน?
✅ Gain + Noise Reduction — GPU (WebGPU) ช่วยได้จริง ประมวลผล sample data แบบ parallel
❌ Encode MP3 — GPU ช่วยไม่ได้ เพราะ lamejs ทำงานบน CPU เท่านั้น ไม่มี MP3 encoder รัน WebGPU ได้ใน browser
